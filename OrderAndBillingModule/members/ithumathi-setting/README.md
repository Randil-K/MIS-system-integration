# ithumathi-setting Module

This directory contains the workspace for ithumathi-setting.

## Frontend
Located in `Order_and_Billing/`. To run:
1. `cd Order_and_Billing`
2. `npm install`
3. `npm run dev`

## Backend
Located in `Order_and_Billing_Backend/`. To run:
1. `cd Order_and_Billing_Backend`
2. `./mvnw spring-boot:run`
